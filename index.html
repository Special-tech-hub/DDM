Project Overview

This project assesses the digital maturity of the University of Zimbabwe across four key dimensions using survey data, unsupervised clustering, and supervised classification.

WiFi Reliability

Campus network performance across the Library, Hostels, and Lecture Rooms — the single strongest predictor of digital maturity.

Emhare Portal

Ease of using the Emhare portal for student registration, results checking, and academic administration.

Clearance Process

Efficiency of the online student clearance process during graduation and semester-end — a shared pain point across all groups.

Google Classroom

Student and staff confidence in using Google Classroom and other digital learning platforms for coursework delivery.

Methodology Pipeline

A structured end-to-end data science workflow from raw survey responses to actionable recommendations.

1

Data Collection

Google Forms surveys distributed to Students, Librarians, and ICT Staff

2

Cleaning

Missing value imputation using column means; column standardisation

3

Clustering

K-Means (k=2) with Elbow + Silhouette validation → Emerging / Established

4

Classification

Decision Tree + Random Forest with 5-fold cross-validation (96.9% accuracy)

5

Insights

Gap analysis, priority matrix, and quarterly improvement roadmap
